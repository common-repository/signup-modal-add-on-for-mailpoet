== Description ==

Das Plugin ermöglicht den Besuchern der Webseite die konfortable Newsletter-Anmeldung über einen frei konfigurierbaren Modal Dialog (Popup).


=== Voraussetzungen ===

* Benötigt MailPoet ab Version 3.
* Optional wird das Plugin CookiePanel empfohlen.

=== Eigenschaften ===

* Frei konfigurierbarer Modal-Dialog mit individuellem Text und Layout
* Es kann eingestellt werden, wann und wie oft der Modal-Dialog zum Anmelden in den Newsletter angezeigt werden soll.
* DSGVO konform - es werden keine Cookies gesetzt
* Erlaubt das DSGVO-konforme Tracken von Statistiken mit Matomo / Google Analytics (setzt das Plugin Cookie Panel voraus)
